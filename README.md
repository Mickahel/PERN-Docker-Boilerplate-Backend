# PERN Docker Boilerplate
a PERN(Postgres, Express, React, Node) Boilerplate for developing and go-prod with Docker

## Technologies
* 🖥️ Node
* 🖼️ React
* 🐋 Docker
* 🤹 Nginx 
* 📊 PM2
* 🐘 PostgreSQL / mySQL

## Platforms
* 🖥️ Backend
* 🖼️ Webapp/App/PWA - Client Side
* 🎛️ Admin - Admin Side
